Qui sommes-nous ?

ControverSciences est conçu pour être accessible à tou·te·s et pour tous·te·s.

Prendre la responsabilité de saisir au vol des débats de notre époque
et les replacer dans un cadre scientifique rigoureux est un défi de taille...
Soyons ambitieux !


Mentions légales
Le site controversciences.org est édité pas l'assocation ControverSciences et al.
Il est intégralement et exclusivement financé par les dons des membres et les prestations réalisées par l'association.
Il n'accepte aucune forme de publicité ni ne reçoit de fonds publicitaire.

Nos partenaires

Université de Montpellier

ControverSciences est utilisé comme outil d'enseignement à l'université de Montpellier depuis Septembre 2015.

Projet AREN

ControverSciences participe au projet AREN (ARgumentation Et Numérique), projet financé par le programme d'investissement d'avenir et piloté par la Caisse des Dépots et Consignations

Si vous aussi vous êtes interessé pour utiliser ControverSciences dans vos cours, nous serons ravis de consolider la passerelle avec vous.

Et derrière le décor ?

Slack

Slack est une plate-forme de communication collaborative ainsi qu'un logiciel de gestion de projets. C'est un peu le forum ou se retrouve l'équipe de ControverSciences !

GitHub

GitHub est un service web d'hébergement de code, utilisant le logiciel de gestion de version git. Le code de ControverSciences est disponible sur GitHub sous la licence libre MIT.

Ruby on Rails

Ruby on Rails, est un framework web libre écrit en Ruby. En tant que framework, il propose une structure au programmeur qui lui permet de développer plus vite et plus intuitivement. C'est le framework de développement de ControverSciences.

Jquery

jQuery est une bibliothèque JavaScript libre et multi-plateforme créée pour faciliter l'écriture de scripts côté client dans le code HTML des pages web. Les pages sur ControverSciences sont intéractives grâce à jQuery.

Bootstrap

Twitter Bootstrap est une collection d'outils utiles à la création de sites et d'applications web. On peut le dire, sans Bootstrap ControverSciences serait beaucoup plus moche.

The Noun Project

The Noun Project est une bibliothèque d'images à la disposition des personnes cherchant des symboles graphiques. Les icônes de ControverSciences proviennent de The Noun Project ou on été fait main.

PostgreSQL

PostgreSQL est un système de gestion de base de données relationnelle et objet (SGBDRO). La base de données de ControverSciences utilise PostgreSQL.

Heroku

Heroku est un service de cloud computing de type plate-forme en tant que service. Il permet le déploiement de ControverSciences sur des serveurs.

MailGun

Mailgun est le service de Rackspace permettant de recevoir, d'envoyer et de suivre les messages électroniques. Il gére les emails (envoyés et reçus) avec le nom de domaine @controversciences.org.

Amazon S3

Amazon S3 (Amazon Simple Storage Service) est un site d'hébergement de fichiers offert par Amazon Web Services. Il stocke les images de ControverSciences.

CloudFlare

CloudFlare est un CDN (service de diffusion de contenus) et un service de serveurs de noms de domaines distribués. Il améliore les performances, la sécurité et la rapidité de ControverSciences.

Pexels

Images et vidéos libres de droit. Les images de l'en-tête des controverses écloses proviennent de Pexels.

Sci-Hub

Sci-Hub rend les publications scientifiques accessibles à tou·te·s en se connectant aux journaux depuis les bibliothèques des universités. Sci-Hub permet ainsi aux utilisateurs de ControverSciences de remonter aux sources des publications scientifiques.
